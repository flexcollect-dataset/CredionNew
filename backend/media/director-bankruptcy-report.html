About the National Personal Insolvency Index (NPII)
About the Index
The NPII is a publicly searchable database maintained by the Australian Financial Security Authority (AFSA). It includes information about:
  • Bankruptcies
  • Debt agreements (Part IX)
  • Personal insolvency agreements (Part X)
  • Deceased estates
The index is updated daily and includes both current and historical insolvency records. Personal insolvency information generally remains on the index permanently, even after discharge.
Search Methodology
This search used exact match criteria for the individual's full name and date of birth. It included:
  • All current insolvency proceedings
  • All historical insolvency records
  • Records with and without middle names
  • Records with and without dates of birth
Parameters were set to capture the broadest possible range of records that could match the individual's details.
